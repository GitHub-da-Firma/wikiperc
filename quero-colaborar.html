Colaborar com a estrutura do site

A WikiPerc é um projeto da Firma, comunidade de projetos de tecnologia e arte. O código fonte se encontra em nosso GitHub e a hospedagem é feita pelo serviço do GitHub Pages. Pretendemos manter a hospedagem do site dessa forma, pois o serviço é gratuito e permite a visita para um público menor de pessoas.

Para quem já tem experiência com Web Development, o código-fonte do site pode ser encontrado no GitHub.

Se você não tem experiência com isso e quer aprender como montar uma página web, recomendamos o curso gratuito de HTML/CSS do Khan Academy.
